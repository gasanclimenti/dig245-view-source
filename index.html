DIG 245 View Source

How to View Source Code (on a Windows Computer):

  1. Open up the Chrome browser.
  2. Find a web page whose source code you would like to view.
  3. Right-click on the page and select "View page source," or use the shortcut "ctrl+U".
  4. A new tab with the code should appear!

What to do with Dev Tools:

  • The "Elements" section allows you to edit the CSS code and see how the updates impact your page in real time.
  • In the "Console" section, you can check and edit Javascript for the webpage.
  • You can send secret messages in the "Console" code!

Metaphor

Looking at the code is like pulling back the curtain. On the presentation-ready stage, much like a webpage, everything seems to run smoothly and seamlessly. Behind the scenes is, to an untrained eye, chaos. But for those who understand the workings of HTML and other code, inspecting a webpage and revealing what's behind the curtain can be informative in how the production actually runs (not by magic).
